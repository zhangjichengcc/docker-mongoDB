# Docker MongoDB

> 部署于docker的MondoDB服务

## How to use

``` bash
# 查看命令
bash run.sh

# 创建容器
bash run.sh start
```

## 文件目录

``` bash
.
├── LICENSE
├── README.md
├── data/                 # 用于存储 MongoDB 数据的本地目录
├── .env                  # 用于存储环境变量
├── docker-compose.yaml   # 镜像配置文件
└── run.sh                # 启动脚本
```
